# food-Delivery-app

A responsive, modern frontend web application for ordering authentic Indian cuisine. This project focuses on delivering a premium user interface with a unique health-conscious twist: real-time nutritional macro tracking.

## 🚀 Live Demo
https://aakashfromujjain.github.io/food-Delivery-app/

## ✨ Key Features
* **Real-Time Macro Tracking:** Automatically calculates and displays the total protein and fiber in the user's cart as they add items to their order.
* **Dynamic Multi-Filtering:** Users can search by text, click category tabs (North Indian, South Indian, Biryani, Desserts), and toggle "Veg Only"—and all filters work flawlessly together.
* **Modern UI/UX:** Features a glassmorphism header, smooth hover animations, and a responsive CSS Grid layout that looks great on both desktop and mobile devices.
* **Instant Cart Updates:** Visual feedback and CSS pop animations trigger instantly when adding items to the cart without requiring page reloads.

## 🛠️ Tech Stack
This project was built entirely with the core technologies of the web, avoiding heavy external libraries to ensure maximum performance and fundamental understanding of DOM manipulation:
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom variables, CSS Grid, Flexbox, and keyframe animations.
* **JavaScript:** Array manipulation, event handling, and complex state filtering.

## 💻 How to Run Locally
Since this is a vanilla web project, no complex installation or build tools are required.
1. Clone the repository: `git clone https://github.com/yourusername/creative-white.git`
2. Open the folder on your local machine.
3. Double-click the `index.html` file to open it in your default web browser.

## 📂 Project Structure
* `index.html` - The main semantic structure and UI layout.
* `style.css` - All styling, theme variables, and animations.
* `script.js` - The data array, filtering logic, and macro calculation engine.
